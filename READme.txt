Program służy do przewidywania cen mieszkań przy użyciu biblioteki scikit-learn i modeli: Decision Tree, Boosted Decision Tree, Random Forest. Dodatkowo zaimplementowałam bibliotekę Extreme Gradient Boosting (XGBoost) i użyłam gradient boosting trees algorithm.
Dane, które przetwarzały modele musiały zostać przeze mnie odpowiednio przygotowane, czyli usunięcie bądź uzupełnienie braków danych, zamienienie zmiennych
przechowujących tekst na zmienne liczbowe oraz standaryzacja zmiennych.
